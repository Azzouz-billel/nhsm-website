# Portfolio Website Project Instructions

## Project Overview
This project is an excellent, premium portfolio website. It leverages modern web technologies with a focus on rich aesthetics, dynamic interactions, and high performance. 

## Commands
```bash
# Development
npm run dev        # start local development server

# Build
npm run build      # create production build

# Styling
npm run lint       # check for style and format errors
```

## Architecture
- `index.html` - Core HTML structure.
- `index.css` - Core design system, variables, and styling. Avoid Tailwind unless specified. Use Vanilla CSS with modern features.
- `src/` or `js/` - Logic, animations, and interactive components.
- `assets/` - Images, 3D models, fonts, and other media.

## Design Guidelines & Aesthetics
- **Rich Aesthetics**: Utilize curated, harmonious color palettes, sleek dark modes, and modern typography (e.g., Inter, Roboto, Outfit).
- **Dynamic Design**: Implement smooth gradients, hover effects, and micro-animations for an alive, premium feel. 
- **SEO Best Practices**: Ensure semantic HTML, proper title tags, meta descriptions, and unique IDs.
- **No Generic UI**: Avoid basic mvp looks. Strive for a "wow" factor at first glance.

## Workflow
- Focus on building a solid foundation in CSS tokens before assembling components.
- Validate animations and responsive layouts consistently across devices.
- Generate high-quality assets if needed instead of using plain placeholders.
